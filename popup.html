Help & Shortcuts

Quick start

Canvascope works best when the extension is pinned and your LMS tab has had a chance to sync once.

  1. Pin Canvascope for one-click access from the toolbar.
  2. Press Cmd/Ctrl + K on a Canvas page to open the quick search overlay.
  3. Tap Refresh Now once if you just opened a class and want a fresh sync.
  4. Try a search from the examples below to learn how the index behaves.

Shortcuts

Popup and overlay controls share the same search engine, but the overlay stays keyboard-first.

Cmd/Ctrl + KOpen the page overlay
EnterOpen the highlighted result
Arrow keysMove through overlay results
EscapeClose Help, Settings, or the overlay
